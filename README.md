1️⃣ var, let, and const

* var: The old way to declare variables. Not recommended to use anymore.
* let: Used when the value will change later.
* const: Used when the value is fixed and will never change.

2️⃣ Spread operator (...)
* It is used to copy or unpack data. It easily copies an old array or object into a new one. Like [...oldArray, newItem].

3️⃣ map(), filter(), and forEach()
* forEach(): Just loops through items one by one. Does not return a new array.
* filter(): Loops and creates a new array with only the items that pass a specific condition.
* map(): Loops and creates a new array by modifying or formatting the original items.

4️⃣ Arrow function (=>)
* A shorter, modern way to write a function. There is no need to type the word function.

5️⃣ Template literals ( )
* Using backticks (`) instead of normal quotes. It allows putting variables directly inside a string using ${variableName}. Very helpful for writing multi-line strings or HTML inside JavaScript.
